Position relative

  1. Una capa arriba de los elementos que tienen posicionamiento static
  2. Los demas respetan su lugar
  3. Se mueve con relacion a su posicion original
Box2

Position absolute

  1. Una capa arriba de los elementos que tienen posicionamiento static
  2. Los demas no respetan su espacio
  3. Se mueve con relacion al contenedor padre que tenga un posicionamiento DIFERENTE a static
  4. Adapta el tama単o al contenido
Box3

Position fixed

  1. Se queda fija en el viewport con su posicion origal
  2. Una capa arriba de los elementos que tienen posicionamiento static
  3. Los demas no respetan su espacio
  4. Se mueve con relacion al viewport
  5. Adapta el tama単o al contenido
Box4

Float Left

  1. Una capa arriba de los elementos que tienen posicionamiento static
  2. Los demas no respetan su espacio, pero el texto si
  3. Se mueve lo mas que puede a la izquierda de su contenedor
  4. Adapta el tama単o al contenido
Box5

Float Right

  1. Una capa arriba de los elementos que tienen posicionamiento static
  2. Los demas no respetan su espacio, pero el texto si
  3. Se mueve lo mas que puede a la derecha de su contenedor
  4. Adapta el tama単o al contenido
